# Jimmy Adams — Personal Page

A clean, travel-inspired personal homepage.

## Setup with GitHub Pages

1. Create a new repository on GitHub named `<your-username>.github.io`
2. Push this code to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://<your-username>.github.io`

## Customize

- Edit `index.html` to update your personal info, links, and content
- Edit `style.css` to change colors, fonts, and layout
- Edit `script.js` to modify animations and interactions

## Tech

- Pure HTML/CSS/JS — no build tools needed
- Responsive design
- Smooth scroll animations
- Google Fonts (Playfair Display, Inter, Caveat)
